ℹ️ Informations Techniques

Théorie des Couleurs Triadiques

Les palettes triadiques utilisent trois couleurs espacées de 120° sur le cercle chromatique :

  • Primaire (200°): Cyan-bleu (Sarcelle) - Couleur de base
  • Secondaire (320°): Rose/Magenta - À 120° de la primaire
  • Tertiaire (80°): Jaune/Vert - À 240° de la primaire (ou -120°)

Utilisation des variables CSS

Chaque palette dispose de :

  • 11 nuances (50-950) avec formules de luminosité et chroma
  • Alias sémantiques (--primary, --secondary, --tertiary)
  • Alias fonctionnels (--bg, --text, --border, --accent)
  • Variations avec couleurs relatives (pastel, accent, muted)

Exemples d'utilisation

/* Utilisation simple */
.element {
    background-color: var(--secondary-500);
    color: var(--secondary-text);
}

/* Utilisation avec alias sémantique */
.button {
    background-color: var(--tertiary-accent);
    border: 1px solid var(--tertiary-border);
}

/* Utilisation avec couleurs relatives */
.custom-element {
    background-color: var(--secondary-pastel);
}
