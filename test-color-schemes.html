📝 Comment utiliser

Modifiez simplement les variables CSS dans votre fichier :

:root {
  --secondary-color-modifier: 120;  /* Triadique */
  --tertiary-color-modifier: 240;   /* Triadique */
  
  /* Ou pour un schéma complémentaire : */
  /* --secondary-color-modifier: 180; */
  /* --tertiary-color-modifier: 30; */
}
